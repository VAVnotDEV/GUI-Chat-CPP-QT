Server Chat GUI

Графическая серверная часть чата на C++.

Проект основан на моём консольном чате:
https://github.com/VAVnotDEV/ConsoleChat-TCP

Основные особенности

Сервер чата с графическим интерфейсом

Поддержка регистрации и авторизации пользователей

Приватные сообщения и общий чат

Управление пользователями (kick / ban)

Отображение списка пользователей и сообщений на сервере

База данных

Используется SQLite

База создаётся автоматически при первом запуске

Таблицы:

users — пользователи

messages — сообщения

Общий чат

Общий чат реализован как специальный контакт common.
Сообщения, отправленные пользователю common, считаются сообщениями общего чата.

Контакт common создаётся автоматически при инициализации базы данных.

Интерфейс

Интерфейс сервера реализован с помощью Qt (Widgets)

Обновление списка пользователей и сообщений происходит автоматически

Сборка и запуск

Проект собирается с помощью qmake.
И является кроссплатформенным.
