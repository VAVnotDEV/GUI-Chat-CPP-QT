GUI Chat (Server + Client)

Проект чата на C++ с использованием TCP-сокетов и SQLite.

Графическая часть сервера реализована с помощью Qt (Widgets).
За основу взят мой консольный TCP-чат:
https://github.com/VAVnotDEV/ConsoleChat-TCP

Структура проекта
Сервер

Серверная часть с графическим интерфейсом

Управление пользователями (список, kick, ban)

Просмотр приватных и общих сообщений

Реализован на C++ + Qt

Клиент

Клиентская часть реализована отдельно

Подключается к серверу по TCP

Поддерживает регистрацию, авторизацию, приватный и общий чат

Логика клиента основана на консольном TCP-чате

База данных

Используется SQLite

База создаётся автоматически при первом запуске сервера

Таблицы:

users — пользователи

messages — сообщения

Общий чат

Общий чат реализован как специальный контакт common.

Сообщения, отправленные пользователю common, считаются сообщениями общего чата.
Контакт common создаётся автоматически при инициализации базы данных.

Запуск сервера (ВАЖНО)

После запуска серверного приложения сокеты не активны по умолчанию.

Чтобы сервер начал принимать подключения, необходимо:

Запустить серверное приложение

В меню выбрать Enable Server

Только после этого сервер начинает слушать входящие соединения

Без этого шага клиенты не смогут подключиться.

Интерфейс

Интерфейс сервера реализован с помощью Qt Widgets

Список пользователей и сообщений обновляется автоматически

Отображается текущий статус сервера (Run / Stop)

Сборка и запуск

Проект собирается с помощью qmake.

После сборки:

Запусти сервер

Нажми Enable Server в Меню

Подключай клиентов
